# Example: Save Output

Below is an example of what `/handoff save` produces for a typical project.

## Command

```bash
# Deno
deno run --allow-read --allow-write --allow-run scripts/save.ts

# Node.js
node scripts/node/save.mjs

# With language and verbosity options
deno run --allow-read --allow-write --allow-run scripts/save.ts --lang zh --verbosity high
node scripts/node/save.mjs --lang en --verbosity low
```

## Mode Differences

### compact

Minimal output - only goal, status, and next steps. Commits limited to 3, no TODO scan.

### full

Maximum context - 20 commits, up to 50 TODO items, full diff stats, risk analysis.

### diff

Focus on changes - includes diff summary, 5 commits, no TODO scan.

## Verbosity Differences

| Feature | low | med | high |
|---------|-----|-----|------|
| Commit count | 3 | 5 | 20 |
| TODO scan | ✗ | ✓ (max 20) | ✓ (max 50) |
| Risk analysis | ✗ | ✓ | ✓ (extended) |
| Diff stats | ✗ | ✓ | ✓ (per-file) |
| tasks.md | ✗ | ✓ | ✓ |
| decisions.md | ✗ | ✓ | ✓ |
| analysis.md | ✗ | ✗ | ✓ |
